🌍 使用說明

以下是如何使用本網站進行匯率換算：

步驟一：輸入資料

  • 輸入項目名稱： 輸入您要換算的項目名稱，例如「耳機」、「機票」、「遊戲點數」等。
  • 輸入價格： 輸入該項目的價格（數字，不需加幣別符號）。
  • 選擇原始幣別： 從下拉選單中選擇幣別，支援：
    • 歐元（EUR）
    • 日圓（JPY）
    • 台幣（TWD）
    • 美金（USD）
  • 選擇支付方式： 包含：
    • 台幣
    • 信用卡
    • 預購

步驟二：新增資料

完成以上資訊後，請點選「新增」按鈕，即可將該筆資料加入清單。

步驟三：查看換算結果

每筆資料新增後，系統會顯示以下資訊：

  • 項目名稱
